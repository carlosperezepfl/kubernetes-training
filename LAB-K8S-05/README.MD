## Using Deployments
 
- Create a deployment inside your namespace using the following specification:


``` yaml
----
apiVersion: v1
kind: Deployment
metadata:
  name: my-app-deployment
  namespace: <YOURNAMESPACE>
spec:
  replicas: 1
  selector:
    matchLabels:
      app: my-app
  template:
    metadata:
      labels:
        app: my-app
    spec:
      containers:
        - name: my-simple-todolist
          image: <TBD>
          ports:
          - containerPort: 8080
```
